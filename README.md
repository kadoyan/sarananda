# 沙羅何蛇 - SARANANDA -
かつて、MSX・FAN誌に掲載されていた一画面プログラム「さらなんだ」をヒントに、[PICO-8で作成したもの](https://www.lexaloffle.com/bbs/?tid=43641)を拡張して作成しました。

🎮[Play(http://kadoyan.github.io/sarananda/sarananda.html)](http://kadoyan.github.io/sarananda/sarananda.html)

- Left and Right: Going left and right
